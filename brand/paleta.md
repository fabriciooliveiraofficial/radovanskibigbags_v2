# Identidade Visual — Radovanski Big Bags

Extraída do logo oficial (folhas sobre arco).

## Cores

| Uso | Cor | Hex |
|---|---|---|
| Primária (verde folha) | Verde agro | `#2E7D32` |
| Secundária (laranja folha) | Laranja/âmbar | `#F5A623` |
| Escura (folhas internas) | Cinza grafite | `#2B2B2B` |
| Preto (folha central / arco) | Preto | `#0A0A0A` |
| Detalhe (caule) | Marrom escuro | `#5C4A07` |
| Fundo | Branco | `#FFFFFF` |
| WhatsApp (CTAs de conversão) | Verde WhatsApp | `#25D366` |

## Aplicação na loja
- Verde `#2E7D32`: cabeçalho, links, elementos de marca.
- Laranja `#F5A623`: destaques, preços, badges de condição (novo/lavado/sujo).
- Grafite `#2B2B2B`: textos e títulos.
- Verde WhatsApp `#25D366`: exclusivo para botões de ação (orçamento/contato) — não misturar com o verde da marca.
- Estilo geral: limpo, fundo branco, sem gradientes/efeitos, tipografia legível (B2B).
